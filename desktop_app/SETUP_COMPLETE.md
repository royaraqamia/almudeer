## 🎨 Customization Tips

### Change App Name
Edit `pubspec.yaml`:
```yaml
name: your_app_name
description: "Your description"
```

### Change Window Size
Edit `lib\main.dart`:
```dart
size: Size(1920, 1080),  // Your preferred size
minimumSize: Size(1024, 768),  // Minimum allowed
```

### Change Theme Colors
Edit `lib\main.dart`:
```dart
colorScheme: ColorScheme.fromSeed(
  seedColor: Color(0xFF000000),  // Your brand color
  brightness: Brightness.light,
),
```
